# 页面模板库
> 能快速找到 高质量,可复用的代码。

将代码按行业分类。行业包括：购物(淘宝，京东)，外卖，资讯(今日头条，知乎)，理财(记账)，旅行(买票，攻略)，其他。

项目，页面，组件的属性包括：行业，设备（PC，手机或都支持）。

每个行业下面包含：项目，页面，组件。项目由页面组成，页面由组件组成。

## 组件规范
* 不污染其他组件
  * CSS 类名要加组件前缀。
  * JS 中添加全局变量，Stroge 等要加组件前缀。
