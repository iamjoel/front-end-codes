## CSS 的常用命名
### 状态
* collapsed 收拢的
* expanded 展开的
* current 当前
* selected 选中的
* active 活跃的
* enabled 可用
* disabled 不可用
* 尺寸。 xl,lg,md,sm,xs 分别对应 特大,大，中，下，特小。
* primary 主要的

## 布局
* 头部 header
* 尾部 footer
* 主栏 main
* 侧栏 side
* 盒容器 wrap
* 条目 item。一般和模块名一起用，如 `list-item`， `news-item`。
